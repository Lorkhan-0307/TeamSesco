How this program works.


1. From GUI, if you choose Webcam, object detection from yolov5 while live camera env activates.
2. From GUI, three pre-recorded video is fully activated with our program. available with following functions.
	- Object detection
	- Check if Object's binding box > 0.3
	- Warning sfx, TTS(Caution), TTS(Object names on list)
	- Record while Warning

3. From GUI, there are two save folders.
	- Record folder runs/detect/(exp[n]) includes fully recored file
	- Record folder runs/savedanger includes Warning record file
